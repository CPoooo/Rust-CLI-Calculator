## How to Run

1. Clone the repo:

```bash
git clone https://github.com/CPoooo/Rust-CLI-Calculator.git
cd Rust-CLI-Calculator
```

2. Run with Cargo:

```bash
cargo run
```

Follow the on-screen prompts to enter two numbers and select an operation.
